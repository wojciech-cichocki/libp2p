# libp2p

```sh
docker-compose up -d
```

# Setup and run manually
```sh
$ cd app
$ npm install
```
```sh
$ cd signaling-server
$ npm inistall
```

```sh
$ cd signaling-server/src/server
$ node index
```

```sh
$ cd app
$ npm run start
```
